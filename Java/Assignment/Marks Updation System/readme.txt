1. Information about files:

a. Code reads the file "Stud_Info.txt" which contains the original record of students, it is an input to the program.
b. "Sorted_marks_by_roll.txt" contains the record sorted by roll number, this file is generated by the program.
c. "Sorted_marks_by_name.txt" contains the record sorted by name, this file is generated by the program.

2. There is a command line interface to interact with the program

3. Run the program by the following commands:

javac UpdateSystem.java
java UpdateSystem
